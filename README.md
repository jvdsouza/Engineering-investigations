# Engineering Investigations
## Mission
To learn to design systems and understand Data Structures and Algorithms from how they apply to real world scenarios, with ChatGPT as a thinking partner.

Initially, this was an initiative for interview prep, but it's now become a source for me to document my learnings, explain how I think, and evolve how I approach these problems. The primary goal is no longer priming myself to pass an interview, but to engage in being a better engineer and discover how I learn best, while being consistent.

This approach is much easier for me to digest, allowing me to associate ideas together with other contexts I already know.
